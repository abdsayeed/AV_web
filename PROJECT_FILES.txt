GETTING STARTED
===============

1. Add logo:       src/assets/logo.png
2. Install:        npm install
3. Start:          npm start
4. Open:           http://localhost:4200
5. Customize:      Edit app.component.ts
6. Build:          npm run build
7. Deploy:         See DEPLOYMENT.md
