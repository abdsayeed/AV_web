## 🔄 Navigation Flow

```
User clicks nav link
        ↓
Smooth scroll to section
        ↓
Active link highlighted
        ↓
Mobile menu closes (if open)
```

## 📋 Form Flow

```
User fills form
        ↓
Selects budget tier
        ↓
Checks services
        ↓
Clicks submit
        ↓
Form validation
        ↓
Success message
```

## 🌙 Dark Mode Toggle

```
User clicks moon/sun icon
        ↓
'dark' class added to <html>
        ↓
All colors transition smoothly
        ↓
Icon changes to opposite
```

## 📐 Responsive Breakpoints

```
Mobile:    < 768px   (1 column, hamburger menu)
Tablet:    768-1024px (2 columns, full nav)
Desktop:   > 1024px   (3 columns, full nav)
```

## 🎯 Key Interactions

1. **Navigation**: Click → Smooth scroll
2. **Dark Mode**: Toggle → Theme change
3. **Mobile Menu**: Hamburger → Slide down
4. **Form**: Fill → Submit → Alert
5. **Buttons**: Hover → Scale up
6. **Cards**: Hover → Lift effect
7. **Links**: Hover → Color change
