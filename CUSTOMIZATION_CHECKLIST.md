# Customization Checklist

Use this checklist to customize your Aries Ventures website.

## 🎨 Branding

- [ ] Add your logo to `src/assets/logo.png`
- [ ] Update company name (currently "Aries Ventures")
- [ ] Change primary color in `tailwind.config.js`
- [ ] Update favicon in `src/` folder

## 📝 Content Updates

### Navigation
- [ ] Review navigation items in `app.component.ts` → `navItems`
- [ ] Add/remove sections as needed

### Hero Section
- [ ] Update main headline
- [ ] Modify subheadline text
- [ ] Change CTA button text
- [ ] Update trust indicators (50+ Businesses, etc.)
- [ ] Replace background image URL

### How It Works
- [ ] Customize step descriptions
- [ ] Update time estimates
- [ ] Change step icons if needed

### Services/Pricing
- [ ] Update pricing plans in `app.component.ts` → `pricingPlans`
  - [ ] Plan names
  - [ ] Prices
  - [ ] Descriptions
  - [ ] Badge text
  - [ ] Images
- [ ] Modify CTA banner text

### Ideal Clients
- [ ] Update industries in `app.component.ts` → `industries`
- [ ] Customize comparison table rows
- [ ] Add/remove industry cards

### Templates
- [ ] Update templates in `app.component.ts` → `templates`
  - [ ] Template names
  - [ ] Industry categories
  - [ ] URLs
  - [ ] Images
  - [ ] Features lists
  - [ ] Badges
- [ ] Add your own template screenshots

### Team
- [ ] Update team members in `app.component.ts` → `teamMembers`
  - [ ] Names
  - [ ] Roles
  - [ ] Bios
  - [ ] Photos
- [ ] Add/remove team members

### Contact Form
- [ ] Customize form fields
- [ ] Update website type options
- [ ] Modify budget ranges
- [ ] Change service options
- [ ] Update privacy policy link
- [ ] Configure form submission (add backend)

### Footer
- [ ] Update contact email
- [ ] Update phone number

- [ ] Modify quick links

- [ ] Add social media links
