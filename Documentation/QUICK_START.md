# 🚀 Quick Start Guide

## View Your Website

**Main Website**: http://localhost:4201/
**Login Page**: http://localhost:4201/login
**Register Page**: http://localhost:4201/register

---

## 🎯 Quick Actions

### Navigate Between Pages:
- Click "Login" in navigation → Go to login page
- Click "Sign Up" in navigation → Go to register page
- Click "Back to Home" → Return to main page

### Test Features:
1. **Scroll through sections** - Navigation highlights active section
2. **Click "Select Plan"** - Shows plan details, scrolls to contact
3. **Click "Preview Template"** - Opens modal with template details
4. **Fill contact form** - Submit to see success message
5. **Try login/register** - Test the authentication pages

---

## 📝 Customization

### Change Content:
Edit `src/app/app.component.ts`:
- Line ~30: Pricing plans
- Line ~50: Industries
- Line ~60: Templates
- Line ~80: Team members

### Change Colors:
Edit `tailwind.config.js`:
- Primary color: `#0369a1`
- Navigation: Update in `app.component.html`

### Add/Edit Templates:
See `TEMPLATE_CUSTOMIZATION_GUIDE.md`

## 🔧 Development Commands

```bash
# Start development server

# Build for production
npm run build

# Stop server
# Press Ctrl+C in terminal
```
